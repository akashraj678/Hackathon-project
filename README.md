# 🔐 BioAuth System

A passwordless authentication system using:
- Behavioral Biometrics (Keystroke + Mouse)
- Trust Score Engine
- Anomaly Detection
- Step-Up OTP Authentication

## 🚀 Features
- Dynamic trust scoring
- Real-time anomaly detection
- Email-based OTP verification
- Dashboard analytics

## 🧠 How it Works
- Score ≥ 75 → Direct Login
- 55–74 → Step-Up OTP
- <55 → Anomaly Detection (3 triggers → OTP)
